Backend Requirements
Python:

Version 3.7 or above
Python Packages:

Flask: pip install flask
Flask-CORS: pip install flask-cors
psycopg2: pip install psycopg2
Frontend Requirements
Node.js:

Version 14.x or above
npm:

Version 6.x or above
React.js:

React and React-DOM libraries (typically included in the package.json dependencies)
Database Requirements
PostgreSQL:
Version 10.x or above
